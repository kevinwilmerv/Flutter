# Financial Management Application – Simpang Raya

This project is a financial management application developed during my internship at **Simpang Raya**.  
The system is designed to help manage business financial activities such as sales tracking, expense recording, and profit analysis.

I am very grateful for the opportunity given by Simpang Raya to contribute and grow through this project.

---

## Features

- Sales and revenue dashboard
- Input data pembelian (expenses tracking)
- Input penjualan & beban usaha
- Perhitungan dan penetapan laba bersih
- Grafik omzet berdasarkan rentang waktu
- Update income per item
- Catatan dan aktivitas log
